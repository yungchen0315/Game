# Game

一個純手機、觸控操作的類世紀帝國風格 RTS 小遊戲，玩家對電腦 AI 進行即時戰略對戰。專為手機**橫向**畫面設計。

## 如何開始遊玩

整個遊戲都寫在單一檔案 `index.html` 裡，用手機瀏覽器直接打開 `index.html` 就能玩，不需要安裝任何東西、也不需要開伺服器。請將手機**橫向**持拿——直向時畫面會提示旋轉裝置。

## 操作方式（純觸控）

- **單指拖曳**：平移視角（跟大多數手機帝國類遊戲一樣，這是最主要的操作）
- **單指輕點**：選取單位／建築；若已選取自己的單位，輕點別處會下指令（移動／採集／攻擊／進駐農田）
- **單指按住不放一小段時間再拖曳**：框選多個單位
- **雙指拖曳**：平移視角（備用手勢，效果跟單指拖曳一樣）
- 選取村民後，下方面板會出現「建造建築物」選項；選取建築後會出現訓練單位／科技研究／時代升級等選項；建造模式下畫面上方會出現「取消建造」按鈕
- **建築物會自動施工完成**，下單、放置後即開始倒數計時完工，不需要村民在旁邊等待，也不會因為誤觸而中斷

## 遊戲內容

- 開局可選擇 6 種文明（各有專屬加成）、1~3 位電腦對手（自由對戰）與難度
- 採集木材/食物/黃金資源、A* 尋路；建築物放置後自動施工完成，不需要村民協助
- 農場建好後可指派一位村民進駐（點農場或用面板按鈕），之後會自動定期產出食物，不用再手動來回搬運
- 城鎮中心/房屋/兵營/射箭場/馬廄/鐵匠鋪/農場/市場/大學/修道院/攻城作坊/城牆/城門/箭塔/奇觀等建築
- 村民、步兵、長槍兵、弓兵、騎兵、僧侶、攻城武器等兵種與兵種升級
- 黑暗/封建/城堡/帝王四個時代科技樹，鐵匠鋪與大學科技研究
- 市場可用木材/食物與黃金互相交易
- 修道院訓練僧侶：治療友軍、感化敵方單位、撿拾地圖上的聖物換取穩定金收入
- 城牆與可開關的城門、具自動攻擊能力的箭塔
- 建成奇觀並撐過倒數時間即可達成奇觀勝利；消滅所有對手則征服勝利
- 戰爭迷霧：只會顯示自己探索過的地形與目前視野內的敵方單位
- 小地圖：顯示已探索地形、各玩家單位/建築位置與目前可視範圍，點擊可直接跳轉視角
- 具備經濟發展、造兵、科技研究、集結進攻決策的電腦對手 AI（多個電腦對手可自由對戰、互相攻打）
- 相機依手機螢幕大小自動縮放，確保視野夠寬廣；下方指令面板在沒有選取任何東西時會自動收合，讓地圖畫面盡量佔滿螢幕
